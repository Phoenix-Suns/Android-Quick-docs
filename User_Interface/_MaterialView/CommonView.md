# Material View Chuẩn

## MaterialButton

```xml
<com.google.android.material.button.MaterialButton
    app:backgroundTint="#fff"
    app:cornerRadius="8dp"
    app:rippleColor="#3F51B5"
    app:strokeColor="#5fed06"
    app:strokeWidth="2dp" />
```

- Màu: backgroundTint
- Màu khi ấn: rippleColor
- Viền: stroke
